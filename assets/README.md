# Dossier Assets

Placez vos fichiers média dans ce dossier :

## Logo
- **Nom du fichier** : `logo.png`
- **Emplacement** : `assets/logo.png`
- **Recommandations** :
  - Format : PNG avec transparence (de préférence)
  - Taille recommandée : 200x200px minimum
  - Le logo sera automatiquement redimensionné à 50x50px dans la navbar

## Vidéo de fond
- **Nom du fichier** : `video-grillarde.mp4`
- **Emplacement** : `assets/video-grillarde.mp4`
- **Recommandations** :
  - Format : MP4 (H.264)
  - Résolution : 1920x1080 (Full HD) ou supérieure
  - Durée : La vidéo sera lue en boucle
  - Poids : Optimisez la vidéo pour le web (compression recommandée)

## Note
Si vous utilisez d'autres noms de fichiers, modifiez les chemins dans `index.html` :
- Ligne 17 : `<img src="assets/logo.png" ...>`
- Ligne 40 : `<source src="assets/video-grillarde.mp4" ...>`
